---
description: Démarrer l'application (Backend et Frontend)
---

Pour lancer l'application, suivez ces étapes :

1. Ouvrez un terminal et allez dans le dossier `backend` :
   ```bash
   cd backend
   npm start
   ```

2. Ouvrez un autre terminal et allez dans le dossier `frontend` :
   ```bash
   cd frontend
   npm run dev
   ```

3. Accédez à l'application via l'URL indiquée par Vite (généralement `http://localhost:5173`).
